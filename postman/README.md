# Using Postman

Test ROMA API using Postman REST client.

## Import Collection

Download the Postman collection:

[Download ROMA API Collection](roma-api-collection.json)

Import into Postman:
1. Open Postman
2. Click "Import"
3. Select downloaded JSON file
4. Collection appears in sidebar
