# Environment Setup

Configure Postman environment for ROMA API.

## Create Environment

1. Click "Environments" in sidebar
2. Click "+" to create new
3. Name it "ROMA API"

## Add Variables

| Variable | Value |
|----------|-------|
| base_url | https://api.izieroma.xyz |
| api_key | sk_live_your_key |
| execution_id | (leave empty) |

## Select Environment

Click dropdown in top-right and select "ROMA API"

## Usage

Variables are referenced as: {{base_url}}, {{api_key}}
