# FAQ

Frequently asked questions about ROMA API.

## General

**Q: Is ROMA API free?**
A: Yes, ROMA is a free community service.

**Q: How do I get an API key?**
A: Email iziedking17@gmail.com or DM @Iziedking on X.

**Q: What are the rate limits?**
A: 1000 requests per day, 10 concurrent executions.

## Technical

**Q: What programming languages are supported?**
A: Any language with HTTP support. We provide guides for Python, JavaScript, React, Next.js, Vue.js, and React Native.

**Q: How long does an execution take?**
A: Usually 10-60 seconds depending on task complexity.

**Q: Can I cancel a running execution?**
A: Yes, use the cancel endpoint.

## Troubleshooting

**Q: Why am I getting 401 errors?**
A: Check your API key is correct and active.

**Q: Why am I getting 429 errors?**
A: You've exceeded the rate limit. Wait for reset or contact us for higher limits.

**Q: My execution is stuck?**
A: Executions timeout after 5 minutes. Check status or contact support.
